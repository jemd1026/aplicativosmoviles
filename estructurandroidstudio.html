🔄 ¿Cómo Trabajan Juntos?

1️⃣ GRADLE construye tu proyecto

Lee build.gradle.kts → Descarga dependencias → Compila código Kotlin/Java

2️⃣ RECURSOS se empaquetan

Toma imágenes de drawable/ → Layouts de layout/ → Textos de values/

3️⃣ MANIFEST configura la app

Define permisos → Registra actividades → Establece punto de entrada

4️⃣ SE GENERA el APK

Todo se empaqueta en un archivo .apk listo para instalar

💡 Puntos Clave :

  • app/ = Donde programarás (90% del tiempo aquí)
  • Gradle Scripts/ = Configuración (solo al inicio del proyecto)
  • MainActivity = Tu primera pantalla a programar
  • activity_main.xml = El diseño de esa pantalla
  • AndroidManifest.xml = Cada nueva actividad debe registrarse aquí

⚠️ ¡No Toques Estos Archivos al Principio!

  • local.properties - Se genera automáticamente
  • gradle-wrapper.properties - Configuración automática
  • proguard-rules.pro - Solo para apps avanzadas

Enfócate en: MainActivity, activity_main.xml, y strings.xml

🎯 Resumen para Principiantes

app/ = Tu código y diseños

Gradle Scripts/ = Configuración automática

¡Programa en app/, deja que Gradle maneje el resto!
